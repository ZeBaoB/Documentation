Constructs a tridiagonal matrix with specified diagonals.

Parameters
dimThe dimension of the matrix (number of rows and columns).
diagPointer to an array containing the main diagonal elements (length dim).
diagSupPointer to an array containing the elements of the superdiagonal (length dim - 1).
diagInfPointer to an array containing the elements of the subdiagonal (length dim - 1).

Populates the matrix such that:

  • The main diagonal is filled with elements from diag.
  • The superdiagonal (above the main diagonal) is filled with elements from diagSup.
  • The subdiagonal (below the main diagonal) is filled with elements from diagInf.
