Detailed Description

Defines the matrix class, a general-purpose matrix class that serves as the parent for other specialized matrix types.

The matrix class provides basic functionality for matrix operations, including element access, modification, and common operations like scaling, addition, and multiplication. It uses a 2D std::vector to store matrix data.
